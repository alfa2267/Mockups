# 2D Mockups

Place your 2D mockup images in this folder.

## Supported Formats
- JPG, JPEG, PNG
- Recommended size: 1200x800px or similar aspect ratio

## Naming Convention
Use descriptive names like:
- logo-concept.jpg
- packaging-design.jpg
- brand-guidelines.png

## Adding to the Site
Update the `mockups` array in `script.js` to include your new 2D mockups:

```javascript
{
    id: unique_number,
    title: "Your Mockup Title",
    type: "2d",
    image: "mockups/2d/your-image.jpg",
    description: "Brief description of the mockup"
}
```
